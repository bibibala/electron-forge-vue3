# electron-forge Vue3 webpack

## 介绍

- 这是一个关于测试 koffi，加载调用 c 库的demo，涉及到加载.node文件，vite还没有解决方案
- https://github.com/vitejs/vite/issues/14289

## install project

- NOTE 不要使用pnpm 因为 electron-forage 不支持

```bash
yarn install
or
npm i
```

## run

```bash
yarn start
or
npm start
```
